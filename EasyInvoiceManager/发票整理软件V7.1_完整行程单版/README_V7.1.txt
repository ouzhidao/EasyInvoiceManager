====================================================
发票整理软件 V7.1 完整版（含行程单功能）
====================================================

【使用说明】
1. 解压本压缩包到任意文件夹
2. 双击运行 "启动V7.1.bat"
3. 首次使用需配置百度OCR API（参考详细使用说明文档）

【新增功能】
- 支持行程单自动识别和整理
- 支持飞机、汽车（滴滴）、火车等类型
- 行程单单独文件夹存放
- Excel新增"行程单"子表
- 滴滴多段行程自动展开

【文件清单】
- 启动V7.1.bat         - 启动程序
- itinerary_parser.py  - 行程单解析器
- data_parser_v2.py    - 数据解析器
- file_organizer_v2.py - 文件整理器
- excel_generator_v2.py- Excel生成器
- gui/main_window_v2.py- 主界面
- gui/print_dialog.py  - 打印对话框

【注意事项】
- 需要Python 3.10+ 环境
- 需要联网使用百度OCR API
- 建议先配置API后再使用
