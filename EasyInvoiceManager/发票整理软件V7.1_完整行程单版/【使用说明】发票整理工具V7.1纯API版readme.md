# 发票智能整理工具 V7.0 纯API版 - 使用说明

## 一、软件简介

本软件用于自动识别和整理电子发票，支持：
- **增值税发票**（电子普通发票、专用发票等）
- **12306铁路电子客票**

输出：
- 自动分类整理到不同文件夹
- 生成Excel清单（含3个子表：增值税发票/铁路客票/识别失败）
- 智能文件名重命名

## 二、使用前准备

### 必须：申请百度OCR API

本软件为**纯API版**，必须配置百度OCR API密钥才能使用。

#### 申请步骤（5分钟）：

1. **访问百度AI开放平台**
   - 网址：https://ai.baidu.com/tech/ocr

2. **登录/注册百度账号**
   - 有百度账号直接登录
   - 无账号需先注册（手机号即可）
   - 注册层面如有不懂，及时询问AI

3. **创建应用**
   - 点击右上角控制台进入
   - 进入控制台后，最上方搜索栏搜索“OCR”，进入子页面后点击应用数的数字，然后点击"创建应用"
   - 填写应用名称（如"发票识别"）
   - 选择"文字识别"-"OCR"
   - 提交创建
   - 注意至少开通，通用文字识别（高精度版）和增值税发票识别，别的例如飞行行程单识别，网约车行程单识别等按需开启（每月有免费额度，建议开启）

4. **获取密钥**
   - 创建成功后，在应用列表中查看
   - 记录三个关键信息：
     - **App ID**
     - **API Key**
     - **Secret Key**

5. **软件中配置**
   - 打开发票整理软件
   - 点击"配置API"按钮
   - 将三个密钥填入对应输入框
   - 点击保存

> **免费额度**：每月1000次调用，对于一般企业使用足够。如不够可购买付费套餐或等次日重置。

## 三、使用步骤

### 第一步：配置API（首次使用必须）

1. 双击 `启动V7.0.bat` 打开软件
2. 点击"配置API"按钮
3. 填入百度API的三个密钥（App ID、API Key、Secret Key）
4. 点击保存
5. **API密钥会自动记忆**，下次打开无需重复配置

### 第二步：整理发票

1. **选择发票来源文件夹**
   - 点击"浏览..."选择存放发票的文件夹
   - 或直接拖拽文件夹到软件窗口

2. **选择输出位置**
   - 点击"浏览..."选择整理后的文件存放位置

3. **点击"开始整理"**
   - 软件会自动：
     - 解压压缩包（如果有）
     - 识别每张发票内容
     - 分类整理（增值税发票/铁路客票/识别失败）
     - 重命名文件
     - 生成Excel清单

4. **查看结果**
   - 处理完成后会弹出统计信息
   - 打开输出文件夹查看整理结果

## 四、输出文件说明

整理完成后，输出文件夹包含：

```
输出文件夹/
├── 增值税发票_总额XXX元/     # 增值税发票PDF文件
├── 铁路电子客票_总额XXX元/   # 铁路电子客票PDF文件
├── 行程单_总额XXX元/         # 行程单PDF文件
├── 识别失败/                 # 无法识别的文件（需人工处理）
└── 发票清单_YYYYMMDD_HHMM.xlsx  # Excel清单（含3个子表）
```

### Excel子表说明：

1. **增值税发票**
   - 序号、新文件名、发票号码、购买方名称、内容、金额、开票方、日期、类型

2. **铁路电子客票**
   - 序号、新文件名、发票号码、购买方名称、乘车人、出发地、到达地、乘车日期、开车时间、席别、金额、开票日期

3. **识别失败**
   - 文件名、失败原因、文件路径

## 五、注意事项

1. **必须联网使用**
   - 本软件使用百度在线API，需要网络连接

2. **API额度限制**
   - 每月1000次免费调用
   - 超出后需购买付费套餐或等次月重置

3. **文件格式支持**
   - PDF、JPG、PNG、BMP、TIFF
   - 压缩包会自动解压（只解压第一层）

4. **识别失败处理**
   - 识别失败的文件会复制到"识别失败"文件夹
   - 可在Excel"识别失败"子表查看原因

## 六、常见问题

**Q: 提示"百度API未配置"怎么办？**
A: 点击"配置API"按钮，填入三个密钥后保存。

**Q: API密钥会泄露吗？**
A: 密钥保存在本地配置文件，不会上传到其他服务器。

**Q: 换了电脑还能用吗？**
A: 需要重新配置API密钥，或复制配置文件到新电脑。

**Q: 每月1000次不够用怎么办？**
A: 可以：1) 购买百度API付费套餐；2) 分多个月处理；3) 等次月免费额度重置。

**Q: 识别准确率如何？**
A: 百度API识别准确率较高，但电子发票中的手写内容、模糊印章等可能影响识别。
