# 发票智能整理工具 V7.0 升级说明

## 主要更新内容

### 1. 双引擎OCR识别
- **PaddleOCR本地识别（优先）**：无需网络，离线识别，保护隐私
- **百度API备用**：当本地识别质量不足时自动降级使用

### 2. 12306铁路电子客票支持
- 自动识别铁路电子客票（电子发票（铁路电子客票））
- 提取关键字段：
  - 发票号码
  - 购买方名称
  - 乘车人姓名
  - 出发地、到达地
  - 车次
  - 乘车日期、开车时间
  - 席别
  - 票价金额
  - 开票日期

### 3. Excel多子表导出
生成的Excel包含三个子表：
1. **增值税发票**：原有的发票汇总表
2. **铁路电子客票**：专门的铁路客票列表
3. **识别失败**：记录识别失败的发票及原因

### 4. 智能文件命名
铁路电子客票命名格式：
```
铁路_{购买方简称}_{乘车日期}_{出发地}-{到达地}_{金额}_{发票后5位}.pdf
```
示例：
```
铁路_莱卡尔食品_20251102_东海县站-连云港站_9.00_38208.pdf
```

### 5. 智能错误处理
- 关键字段缺失检测（出发地/到达地/金额任一缺失算失败）
- 识别失败自动复制到"识别失败"文件夹
- 失败原因记录在Excel"识别失败"子表中

## 安装说明

### 前置要求
- Python 3.8 或更高版本
- Windows 10/11 操作系统

### 安装步骤

1. **安装依赖**
   ```bash
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```

2. **PaddleOCR首次运行**
   - 首次运行时会自动下载OCR模型（约200MB）
   - 下载完成后保存在 `C:\Users\{用户名}\.paddlex\` 目录

3. **配置百度API（可选）**
   - 如果PaddleOCR识别效果不理想，可配置百度OCR API作为备用
   - 点击"配置API"按钮，输入App ID、API Key、Secret Key

## 使用说明

### 启动程序
双击 `启动V7.0.bat` 或在命令行执行：
```bash
python main.py
```

### 操作步骤
1. 选择发票来源文件夹（支持拖拽）
2. 选择输出位置
3. 点击"开始整理"
4. 等待处理完成
5. 查看输出文件夹中的Excel和整理后的发票

### 文件夹结构
```
输出文件夹/
├── 增值税发票_总额XXX元/     # 增值税发票
├── 铁路电子客票_总额XXX元/   # 铁路电子客票（新增）
├── 识别失败/                 # 识别失败的发票（新增）
└── 发票清单_YYYYMMDD_HHMM.xlsx  # 包含三个子表
```

## 故障排除

### PaddleOCR初始化失败
**现象**：程序启动时显示"PaddleOCR初始化失败"
**解决**：
1. 确保Python版本 >= 3.8
2. 尝试重新安装：`pip install paddleocr --upgrade`
3. 如仍失败，配置百度API作为替代方案

### 铁路客票识别失败
**现象**：12306发票被识别为失败
**解决**：
1. 检查OCR引擎状态，优先使用PaddleOCR
2. 确保发票PDF清晰可读
3. 查看"识别失败"子表了解具体原因

### 百度API额度用完
**现象**：提示"QPS超限"或"额度不足"
**解决**：
1. 登录百度智能云控制台查看额度
2. 或等待次日自动重置免费额度
3. 本地PaddleOCR仍可正常使用

## 技术说明

### 识别策略
```
输入文件
    ↓
PaddleOCR本地识别（优先）
    ↓
质量检查（关键字段是否完整）
    ↓
质量好 → 使用本地结果
质量差 + 有API → 降级到百度API
质量差 + 无API → 使用本地结果并提示
    ↓
发票类型判断（增值税/铁路客票）
    ↓
专用解析器提取字段
    ↓
验证关键字段完整性
    ↓
有效 → 整理到对应文件夹
无效 → 复制到"识别失败"文件夹
```

### 铁路客票识别关键词
- "电子发票（铁路电子客票）"
- "买票请到12306"
- "中国铁路"
- 车次格式：K/G/D/C/Z/T/Y/L + 数字
